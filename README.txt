# Racers
### Overview

Racers is a web application designed for car racing enthusiast. The club aims to provide a space for members to showcase their passion for racing, learn from each other, and participate in events and competitions.

## Tech Stack

- Frontend: HTML, CSS, JavaScript, Bootstrap, Razor pages
- Backend: ASP.NET Core
- Database: Microsoft SQL Server

## Getting Started

Clone the repository

```bash
  git clone https://github.com/siya-mngadi/RacersApp.git
```

 Restore Project
```bash
  dotnet restore
```

 Run Project
```bash
  dotnet run
```
